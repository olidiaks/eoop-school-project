# eoop-school-project

# Polish
Jest to projekt na studia na 2 semestrze z przedmiotu Programowanie obiektowe.
Project będzie stopniowo rozwijany. 
Jednakże jest to raczej ciekawostka niż faktycznie użyteczny kawałek kodu.
Jedyne zastosowanie to, jeżeli ktoś potrzebuje przykład jak napisać obiektowo program 
lub jak napisać testy do niego w C++.

# English
This is a school project for 2 semesters of the subject Object-Oriented Programming.
The project will be gradually developed.
However, it isn’t really useful code.
The only use is if someone needs an example of how to write an object-oriented program
or how to write tests to it in C++.
